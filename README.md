# Amazon Clone

A responsive front-end clone of Amazon built using HTML and CSS. This project recreates the look and feel of Amazon's homepage, including the navigation bar, hero section, product categories, and footer.

## Features

* Responsive design
* Amazon-inspired UI
* Navigation bar with search section
* Product category cards
* Hero banner section
* Footer with multiple links

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Purpose

This project was created to practice front-end web development skills, focusing on page layout, styling, responsiveness, and UI design using only HTML and CSS.

## Installation

1. Clone the repository.
2. Open `index.html` in your browser.
